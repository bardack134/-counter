# -counter
## -カウンターボタンプロジェクト

 JavaScriptとHTMLを使ったシンプルなカウンターアプリです。ボタンをクリックすることで、カウンターの値を増減したり、リセットしたり、できます。

 色の変更: 値が正なら青になり、負なら赤になり、ゼロなら黒に色を変更。
